## Liste à puces

^b104ab

Pour créer une liste à puces, le texte doit être précédé d'un - (ou d'un +) suivi d'un espace.
```markdown
- Liste à puces
+ Liste à puces
```
Donne
- Liste à puces
+ Liste à puces

## Liste ordonnée

^de97a2

Pour créer une liste ordonnée, le 1er élément de la liste doit être précédé du numéro "1" suivi d'un point. Par la suite, l'incrémentation peut se faire automatiquement.
```markdown
1. Liste à puces
2. Liste à puces
```
Donne
1. Liste à puces
2. Liste à puces

## Liste de cases à cocher

^ac9380

Les cases à cocher sont réalisées grâce aux crochets. Avec un x dedans, la case est cochée.
```markdown
- [x] Item 1, coché
- [ ] Item 2, non coché
```
Donne
- [x] Item 1, coché
- [ ] Item 2, non coché
