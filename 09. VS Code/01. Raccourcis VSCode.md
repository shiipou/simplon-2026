# Raccourcis essentiels VS Code

## Fichiers et l’interface

- Nouveau fichier : `Ctrl + N`
- Ouvrir fichier : `Ctrl + O`
- Enregistrer fichier : `Ctrl + S`
- Fermer fichier : `Ctrl + W`
- Aller à un fichier : `Ctrl + P`
- Split view / Diviser l’éditeur : `Ctrl + \`
- Basculer entre éditeurs : `Ctrl + Tab`
- Ouvrir paramètres : `Ctrl + ,`

---

## Gestion des onglets et fenêtres

- Fermer tous les onglets : `Ctrl + K`, `Ctrl + W`
- Réouvrir le dernier onglet fermé : `Ctrl + Shift + T`
- Fermer l’éditeur actuel : `Ctrl + F4`
- Ouvrir palette de commandes : `Ctrl + Shift + P`

---

## Terminal intégré

- Ouvrir / fermer le terminal : `Ctrl + J`
- Créer un nouveau terminal : `Ctrl + Shift + ù`
- Basculer entre les terminaux : `Ctrl + Page Up / Page Down`

---

## Recherche et remplacement avancée

- Rechercher et remplacer dans fichier : `Ctrl + H`
- Rechercher et remplacer dans projet : `Ctrl + Shift + H`
- Rechercher dans fichier : `Ctrl + F`
- Rechercher dans projet : `Ctrl + Shift + F`

---

## Formatage et auto-completion

- Formatter le document : `Shift + Alt + F`
- Suggestions / Auto-completion : `Ctrl + Space`
- Auto-import : `Ctrl + .`

---

## Navigation dans le code

- Commenter ligne : `Ctrl + /`
- Supprimer ligne : `Ctrl + Shift + K`
- Déplacer ligne : `Alt + ↑ / ↓`
- Aller à une ligne : `Ctrl + G`
- Copier ligne : `Shift + Alt + ↑ / ↓`
- Renommer symbole : `F2`

---

## Debug

- Démarrer / Continuer : `F5`
- Step over : `F10`
- Step into : `F11`
