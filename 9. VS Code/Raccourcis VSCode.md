# VS Code — Raccourcis essentiels (Ultra minimal)

## Commandes & Navigation
- Palette de commandes : Ctrl + Shift + P
- Aller à un fichier : Ctrl + P
- Aller à une ligne : Ctrl + G
- Paramètres : Ctrl + ,

## Édition
- Commenter ligne : Ctrl + /
- Supprimer ligne : Ctrl + Shift + K
- Déplacer ligne : Alt + ↑ / ↓
- Copier ligne : Shift + Alt + ↑ / ↓ 
- Renommer symbole : F2

## Sélection multiple
- Sélectionner occurrence suivante : Ctrl + D
- Tout sélectionner occurrences : Ctrl + Shift + L
- Multi-cursor : Alt + Click

## Recherche
- Rechercher dans fichier : Ctrl + F
- Rechercher dans projet : Ctrl + Shift + F

## Debug
- Démarrer / Continuer : F5
- Step over : F10
- Step into : F11
