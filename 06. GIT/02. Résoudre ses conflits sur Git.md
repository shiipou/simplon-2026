#### Si on a un nom de fichier exotique :
Ici les guillemets sont très importants car ça va échapper les espaces et ne pas s'arrêter dessus.
Les éléments _\303\250_ correspond à un caractère spécial, qu'il faut remplacer par le caractère adéquat.
  

```

"7. Espace de travail/Probl\303\250mes techniques.md"

```

```

git restore --staged "7. Espace de travail/Problèmes techniques.md"

```
