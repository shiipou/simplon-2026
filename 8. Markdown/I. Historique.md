Le markdown est un langage de balisage léger. Il a été co-crée par Aaron Swartz et John Gruber en 2004.
Le but initial est de pouvoir générer facilement du code HTML pour le web. Depuis, il a été étendu afin de produire d'autres types de documents, parmi lesquels le pdf.

Il existe des variantes telles que *Pandoc*, *MultiMarkdown* ou encore *GitHub Flavored Markdown*.
